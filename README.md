# GoBindFile_fish
一键构建捆绑式文件

## 功能介绍：

<br>
1.执行后，自删除自身<br>
2.无窗口执行，随机生成文件名称<br>
3.隐藏执行恶意文件后释放诱饵文件<br>

## 使用介绍：
```
binding.exe <evil_program> <bind_file>
示例：binding.exe  calc.exe 'WP(1).pdf'
```
